Meanwhile I have come to the conclusion that this syntax for binding arguments
in #JavaScript

const f1 = f.bind(undefined, arg) f1(); // const f = arg =&gt; {}

is inferior to

const f1 = f(arg) f1(); // const f = arg =&gt; () =&gt; {}
