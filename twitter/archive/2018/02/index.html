  • If you're an engineer and need an easy to follow algorithm for giving good #feedback, check out non-violent communication: https://t.co/QWiLvl11Xv It describes 4 steps which enable you to give feedback while reducing the chance to offend the receiver:
  • 1) Observation: describe the facts (what your are seeing, reading) without evaluating them.
  • 2) Express how this impacted you personally. Do not talk about how others might be impacted, or generalize impact.
  • 3) Describe what you need instead, what would help you. Again, no generalization.
  • 4) Als for a specific, concrete action which would help to achieve what you described in 3. This inverses the feedback from being a request/command to a call for help. This is very powerful!
